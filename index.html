Retrieval-Augmented Generation

Developed an AI-powered PDF chatbot using retrieval-augmented generation (RAG) with a Python backend, enabling users to query documents with precise, context-aware responses. Implemented document chunking and vector storage in ChromaDB to efficiently retrieve the top four relevant chunks per query, which are processed by a local LLM running via Ollama. Built a Flask-based frontend with HTML, CSS, and JavaScript, allowing seamless interaction through a chat interface. The application operates entirely on-device, ensuring privacy, fast response times, and no dependency on external servers or APIs, making it ideal for handling sensitive information.

  • ChromaDB
  • Local LLM
